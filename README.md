# MERN Todo App

A full-stack Todo application built using the MERN stack (MongoDB, Express, React, Node.js).

---

## Features

- User Signup & Login
- JWT Authentication
- Add Tasks
- Update Tasks
- Delete Tasks
- View All Tasks
- Protected Backend Routes

---

## Tech Stack

Frontend:
- React
- Vite
- CSS
- React Router

Backend:
- Node.js
- Express.js
- MongoDB
- JWT
- Cookie Parser
- CORS

---

## API Routes

- POST /signup → create user
- POST /login → login user
- GET /tasks → get tasks
- POST /add-task → add task
- PUT /update/:id → update task
- DELETE /delete/:id → delete task
